calibrefx_human_time_diff (line 114)

Calculate the time difference - a replacement for human_time_diff() until it is improved.

Based on BuddyPress function bp_core_time_since(), which in turn is based on functions created by Dunstan Orchard - http://1976design.com

This function will return an text representation of the time elapsed since a given date, giving the two largest units e.g.:

  • 2 hours and 50 minutes
  • 4 days
  • 4 weeks and 6 days

  • return: The time difference
  • since: 1.7.0
str calibrefx_human_time_diff ($older_date $older_date, [$newer_date $newer_date = false])
  • $older_date $older_date: int Unix timestamp of date you want to calculate the time since for
  • $newer_date $newer_date: int Unix timestamp of date to compare older date to. Default false (current time)
calibrefx_truncate_phrase (line 37)

Return a phrase shortened in length to a maximum number of characters.

Result will be truncated at the last white space in the original string. In this function the word separator is a single space (' '). Other white space characters (like newlines and tabs) are ignored.

If the first $max_characters of the string do contain a space character, an empty string will be returned.

string calibrefx_truncate_phrase (string $phrase, integer $max_characters)
  • string $phrase: A string to be shortened.
  • integer $max_characters: The maximum number of characters to return.
